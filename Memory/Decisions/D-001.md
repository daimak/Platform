# Decision D-001 — Git + Branching model

## Context
Проект Platform начал использовать Git как систему версионирования мышления.

## Decision
Используются две ветки:
- main → стабильное состояние системы
- experiment → тестирование новых идей

## Reason
Нужно разделить стабильное мышление и экспериментальное, не усложняя структуру.

## Consequences
- уменьшение риска сломать основную линию мышления
- появление экспериментальной зоны
- сохранение простоты архитектуры
